## 📖 Conteúdo do Curso

O curso aborda os principais conceitos do ecossistema React, incluindo:

- ⚛️ Fundamentos do React (componentes, JSX, props, state)
- 🔄 Ciclo de vida e Hooks (`useState`, `useEffect`, `useContext`, etc.)
- 🗺️ React Router (navegação entre páginas)
- 🗃️ Gerenciamento de estado com Redux e Redux Toolkit
- 🌐 Integração com APIs e requisições HTTP
- 🔐 Autenticação em aplicações React
- ▲ Next.js (SSR, SSG, App Router e muito mais)
- 🚀 Deploy de aplicações React e Next.js
- E muito mais...

## 🚀 Como Executar os Projetos

1. Clone o repositório:

   ```bash
   git clone https://github.com/rhuanlucasdev/estudos.git
   ```

2. Acesse a pasta do projeto desejado:

   ```bash
   cd estudos/01-starting-project
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
